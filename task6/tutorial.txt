Привіт.

Це коротка інструкція до гри The Rotten Heart.
Тут описано механіки гри, які ти зустрінеш протго свого проходження.


    1. Інсталяція (пункт для непрограмістів. ФПН пропускайте)
1) На твоєму комп'ютері має бути встановлений інтерпритатор Python. Він інсталюється з офіційного сайту

Ось інструкція по встановленню
Дізнайтеся розрядність операційної системи (FAQ). Більшість сучасних операційних систем є 64-розрядними (64-бітними).
Завантажте відповідну до вашої розрядності версію
64-бітну версію (www.python.org/ftp/python/3.8.5/python-3.8.5-amd64.exe)
32-бітну версію (www.python.org/ftp/python/3.8.5/python-3.8.5.exe)
Під час встановлення поставте галочку Add Python 3.8 to PATH (додаємо у змінну середовища PATH шлях до директорії де буде знаходитися бінарний файл інтерпретатора для запуску з терміналу)
Оберіть рекомендований варіант встановлення.
Перевірте встановлення - запустіть командний рядок (cmd, якщо ви не знайомі з командним рядком то перегляньте відповідний пункт "Елементи комп'ютерної грамотності", FAQ) чи термінал та виконайте python -V.
Якщо висвітлилася версія Python 3.8.5 ви успішно встановили інтерпретатор.

2) Усі файли гри розархівовуєш у окрему директорію на комп'ютері.
(крім tutorial.txt — він ніяк не впливає на ігровий процесс)
3) Для запуску гри потрібен термінал. Для доступу до нього, у пошуковій стрічці Windows вводиш назву CMD
4) Далі, тобі треба перейти у дирекорію з грою. Повну директорію можна побачити, якщо
у файловому менеджері натиснути на верхнє поле. Копіюй цю стрічку.
5) У терміналі вводиш команду "cd <вставляєш директорію>" (без <>)
6) Далі, щоби запустити гру, вводиш стрічку "python TheRottenHeart.py"
Якщо це не запустило гру, то спробуй "python3 TheRottenHeart.py"
7) Все!


    1. ЗАГАЛЬНІ КОМАНДИ
Для виконання будь-якої дії тобі треба знати основні команди, кожна з яких відповідає
певній дії. Ось їх список:
[що?, шо?, поможіт] — вивести список усіх команд
[північ, південь, захід, схід] — переміщення у відповідному напрямку
[говорити, поговорити, побалакати] — поговорити з союзником (не працює на ворогів)
[битися, бій, битва] — битва з ворогом (не працює на союзників)
[взяти] — взяти предмет, що лежить у локації, в наплічник
[наплічник, інвентар] — переглянути предмети в наплічнику
[вихід, вийти] — вийти з гри


    2. СИСТЕМА БОЮ
Протягом проходження ти стикнешся в битві з декількома ворогами.
Система бою в грі The Rotten Heart працює на реакції.
Після початку бою на екран виводиться відповідне повідомлення. (————— Бій —————)
За цим на екран виводиться повідомлення ГОТУЙСЯ.
Протягом наступних 1-4 секунд на екраніт з'явиться меч. 0===[=========>
Як тільки він з'явився, завдання гравця натиснути на клавішу Enter якнайшвидше.
Якщо реакція була достатньо швидкою, бій виграно. А інакше, бій програно та гра закінчується.

Також, для того щоби почати деякі бої, гравец має мати спеціальні предмети у наплічнику.
Повідомлення про їх відсутність буде виведено на екран.
Такі бої неможливо розпочати, допоки не буде знайдено вказаний предмет.


    3. СОЮЗНИКИ
Протягом проходження ти устрінеш декілької персонажів, з якими можна буде поговорити.
Основна мета цих розмов полягає у просуванні по сюжету та отриманні потрібних предметів.
На відміну від битв, союзники не вказують, який саме предмет тобі слід отримати для
подальшого просування по сюжету.
